使用命令行工具操作文件与目录

发表于 2015-06-15 |

命令行工具是做Web设计与开发的必备工具，通过输入命令，去做不同的事情，比如基本的文件与目录的操作，去安装，更新，卸载软件，控制与管理服务器等等。

准备

不同操作系统上提供了不同的命令行工具，Windows上可以使用系统自带的 命令提示符（cmd）或者Powershell（Win7以上版本，推荐使用），在Mac上用的是系统自带的 终端（Terminal）。

命令提示符

打开开始菜单，在搜索框中输入 cmd，点击结果里的 cmd。

Powershell

打开开始菜单，在搜索框中输入 powershell，点击结果里的 Windows Powershell。

终端

使用 Ctrl + 空格 打开Spotlight中搜索 Terminal。

文件与目录的基本操作

你可以通过命令浏览系统里的目录，创建，移动，删除目录以及文件

查看你的当前位置

Powershell，终端在鼠标光标的左侧会默认显示你当前所在的目录。你也可以使用 pwd 这个命令查看你当前所在的位置。

显示当前目录下的文件：ls

想要查看当前目录下有哪些东西，可以使用 ls 命令，如果在命令后指定一个目录的路径，那么会显示指定路径里的东西，如果直接使用此命令，则会显示你当前坐在目录下的东西。

Windows Powershell会比较详细的显示出目录下的所有东西，比如文件和目录的权限，最近更新的时间等等。

Mac的终端上只会显示出简单的文件和目录名称，想要得到更详细的信息，可以再 ls 命令后加上一个 -l 的参数，另外默认的ls命令的显示结果并不包括该目录下的隐藏文件和文件夹，你需要添加一个 -a 的参数。

在 ls 后可一个加一个路径，这样会显示路径下的文件和目录，可以使相对路径，也可以是绝对路径。

上面这条命令使用了相对路径，它会显示出当前所在目录下的desktop文件夹下的所有东西。

清空命令行上的显示：clear

在屏幕上显示的东西太多了，清空一下，可以使用 clear 这个命令。Windows上也可以使用 cls。

进入到某个目录下面：cd

在图形界面中，想要进入到某个目录下面，可以双击打开这个目录。在命令行工具下，你需要使用 cd 这个命令，后面加上你想要进入到的目录名称。

想要返回上一级目录，可以再cd命令后加 ../。

创建目录：mkdir

在指定的位置创建一个目录，可以使用 mkdir 后面加上你要创建的目录名称。

如果需要创建一个目录结构，比如你想要在某个目录下，在创建一个子目录，可以在mkdir命令后加上需要创建的目录路径，如果这个路径上的目录不存在，则会去自动创建一个。

在Mac和Linux下需要使用一个 -p 的参数。

移动目录与文件：mv

想要将目录或者文件移动到一个新的地方，需要使用 mv 命令。在命令后先是要移动的东西，跟一个空格，接着是要移动到得新位置。

复制目录与文件：cp

cp 命令可以复制指定的文件或者文件夹，先指定一下要复制的东西，空格分割一下，后面跟要复制到的位置。

比如在d盘下新建一个index.md文件，想要把它复制到D盘下的project文件夹下一份。

想要复制文件夹，需要加一个 -R 参数。

删除文件或目录：rm

想要把指定的文件或者目录删除，使用的是 rm 命令，在它后面加上要删除的东西就行了，在删除目录的时候，需要加上一个 -r 参数，另外可以再加上一个 -f 参数，这样在删除目录中的子文件和文件夹时就不会出现提示了，注意在Powershell上使用rm命令不需要参数。
