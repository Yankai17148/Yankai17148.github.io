Web中文字体压缩方案 Font-Spider

发表于 2015-04-08 |

字蛛是一款中文Web-Font压缩工具

特性

  1. 中文字体压缩，可以将几MB的中文字体压缩为十几KB
  2. 用法简单，一段命令即可完成
  3. 文本支持选中，搜索，朗读，翻译，缩放

安装

安装font-spider需要nodejs环境以及npm

Mac下需要管理员权限 sudo 。

使用

在CSS下声明字体文件

Tips

  1. @font-face 中定义的 src 属性中的 .ttf文件必须存在，其他格式的字体工具会自动生成
  2. 不支持动态插入的CSS和字符
  3. CSS content 插入的字符需要定义 font-family ，不支持继承

压缩命令

例子： $ font-spider dist/*.html

劣势

  1. 由于font-spider的运行机制是筛选html文件中使用的字符之后从字体文件中筛选出来删除无用的字符来减小字体文件的大小，所以font-spider并不适用于大段文字以及文章正文
  2. 在开发过程中需要多次编译保证效果
  3. 某些字体会出现编译不通过的情况，待官方修复
